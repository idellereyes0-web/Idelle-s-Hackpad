# Idelle-s-Hackpad
This is my entry for the hackclub hackpad blueprint! My hackpad is simple and is used to adjust volume, open apps, and to copy/paste.
BOM - 4x MX-Style switches, 4x white blank DSA keycaps 1x unsoldered Seeed XIAO RP2040, 1x EC11 Rotary encoder, 1 piece case.
